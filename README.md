## How to Set Up and Run

### Prerequisites
* Node.js
* MongoDB (local or a cloud Atlas string)

---

### 1. Backend Setup

1.  Navigate to the backend folder:
    `cd backend`

2.  Install dependencies:
    `npm install`

3.  Create a `.env` file in `backend/` and add your `MONGO_URI` and `JWT_SECRET`.

4.  Run the backend server:
    `npm run dev`

---

### 2. Frontend Setup

1.  Open a new terminal.
2.  Navigate to the frontend folder:
    `cd frontend`

3.  Install dependencies:
    `npm install`

4.  Run the frontend development server:
    `npm run dev`
